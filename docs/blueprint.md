# **App Name**: CodeCraft Portfolio

## Core Features:

- Project Showcase: Display Python projects with detailed descriptions and links to repositories/live demos.
- Skills Display: Dynamically showcase developer skills with proficiency levels and related technologies.
- About Me Section: Provide an 'About Me' section with a professional bio and personal interests.
- Contact Form: Implement a contact form for visitors to send direct messages.
- Theme Toggle: Add a toggle button on the navbar for switching between dark and light themes.

## Style Guidelines:

- Dark background (#121212) to create a minimalist and professional feel.
- Primary color: Vibrant green (#32CD32) for accents and highlights.
- Accent color: A slightly lighter shade of green (#55FF55) to provide subtle contrast for interactive elements.
- Body and headline font: 'Inter', a grotesque-style sans-serif for a modern look.
- Use minimalist, line-based icons in green to complement the dark theme.
- Clean and organized layout with a focus on readability and ease of navigation.
- Subtle transitions and animations for interactive elements to enhance user experience.
